## Github 코드 올리기
```bash
git push
```
---
## Github 코드 받아오기
```bash
git clone 원격저장소주소
```
---
## Github 코드 가져오기
```bash
git pull 원격저장소주소
```
---
## Github 프로젝트 첫 업로드
1. Your repositories 접속
2. New 버튼 클릭
3. Repository name 입력
4. create repository 버튼 클릭
5. …or create a new repository on the command line 의 마지막 2줄 입력
